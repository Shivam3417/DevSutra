HTML Introduction

HTML ka full form HyperText Markup Language hai. Ye web pages banane ke liye use hoti hai.


HTML Kya Karta Hai?


  • Web page ka structure banata hai
  • Text, images, links ko organize karta hai
  • Browser ko batata hai content ka type kya hai


Example

            <!DOCTYPE html>
            <html>
            <head>
            <title>My Page</title>
            </head>
            <body>
            <h1>Hello World</h1>
            </body>
            </html>

HTML Elements

HTML document different elements se milkar banta hai. Ek element start tag, content aur end tag se milkar banta hai.


Structure


<tagname> Content </tagname>



Example

<h1>This is a Heading</h1>
<p>This is a paragraph</p>


Note:


Some elements empty hote hain jaise <br> aur <hr>.

HTML Attributes

Attributes HTML elements ko extra information dete hain. Ye hamesha start tag ke andar likhe jate hain.


Syntax


name="value"



Example

<a href="https://google.com">Visit Google</a>


Common Attributes


  • href
  • src
  • alt
  • class
  • id

HTML Headings

HTML me 6 heading levels hote hain h1 se h6 tak. h1 sabse important heading hoti hai.



Example

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>

HTML Paragraphs

Paragraph likhne ke liye <p> tag use hota hai. Browser automatically paragraph ke upar aur niche space deta hai.


Line Break


<br> tag line break dene ke liye use hota hai.



Example

<p>This is a paragraph</p>
<br>
<p>Second paragraph</p>

CSS Introduction

CSS ka full form Cascading Style Sheets hai. Ye web pages ko design aur style dene ke liye use hoti hai.


CSS Kya Karta Hai?


  • Colors add karta hai
  • Fonts change karta hai
  • Layout control karta hai
  • Responsive design banata hai


Example

body {
  background-color: lightblue;
}

CSS Syntax

CSS rule selector aur declaration block se milkar banta hai.



Example

p {
  color: red;
  font-size: 18px;
}

CSS Selectors

Selectors batate hain ki kaunse element ko style karna hai.


  • Element selector
  • Class selector (.class)
  • ID selector (#id)

Example

#title {
  color: blue;
}

.box {
  border: 1px solid black;
}

CSS Colors

CSS me colors name, HEX, RGB ya HSL se define kiye jate hain.


Example

h1 {
  color: #ff0000;
}

p {
  color: rgb(0, 128, 0);
}

CSS Box Model

Har HTML element ek box ki tarah hota hai jisme content, padding, border aur margin hote hain.


Example

div {
  margin: 20px;
  padding: 15px;
  border: 2px solid black;
}

JavaScript Introduction

JavaScript ek programming language hai jo web pages ko interactive banati hai.


JavaScript Kya Karta Hai?


  • Button click pe action perform karta hai
  • Form validation karta hai
  • Dynamic content change karta hai
  • Animations create karta hai


Example

<button onclick="showMessage()">Click Me</button>

<script>
function showMessage() {
  alert("Hello DevSutra!");
}
</script>

JavaScript Variables

Variables data store karne ke liye use hote hain.


  • var (old method)
  • let (recommended)
  • const (constant value)

Example

let name = "Rahul";
const age = 20;

console.log(name);

JavaScript Functions

Function ek reusable block of code hota hai.


Example

function greet() {
  console.log("Welcome to DevSutra!");
}

greet();

JavaScript Events

Events tab trigger hote hain jab user koi action karta hai.


  • onclick
  • onmouseover
  • onchange

Example

<button onclick="changeText()">Click</button>

<script>
function changeText() {
  document.body.style.backgroundColor = "yellow";
}
</script>

JavaScript DOM

DOM (Document Object Model) JavaScript ko HTML elements ko access aur modify karne deta hai.


Example

<p id="demo">Hello</p>

<script>
document.getElementById("demo").innerHTML = "Hello DevSutra!";
</script>
