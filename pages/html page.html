HTML Introduction

HTML (HyperText Markup Language) is the standard language used to create the structure of web pages. It defines how content such as text, images, links, and multimedia elements are displayed in a browser.

What is HTML?

HTML is a markup language, not a programming language. It uses elements (also called tags) to structure content. Browsers read HTML files and render them visually for users.

Basic Structure of an HTML Document

Every HTML page follows a standard structure:

<!DOCTYPE html>
<html>
<head>
  <title>Page Title</title>
</head>
<body>
  <h1>Hello World</h1>
</body>
</html>

Key Components

  • <!DOCTYPE> – Defines the document type
  • <html> – Root element
  • <head> – Contains metadata
  • <body> – Contains visible content

Why HTML is Important?

  • Creates webpage structure
  • Supports multimedia
  • Works with CSS & JavaScript
  • Essential for SEO and accessibility

HTML5 Features

Modern HTML5 introduced semantic elements like <header>, <section>, <article>, <footer>, which improve readability and SEO.
