CSS Introduction

CSS (Cascading Style Sheets) is used to control the appearance and layout of web pages. It makes websites visually appealing and responsive.

What Does CSS Do?

  • Changes colors and fonts
  • Controls spacing and alignment
  • Creates layouts
  • Adds animations and transitions

Basic CSS Syntax

selector {
  property: value;
}

Example:

h1 {
  color: blue;
  font-size: 30px;
}

How CSS Works

CSS works by selecting HTML elements and applying styles to them. It follows the "cascade" rule, meaning the last applied rule overrides previous ones.

Types of CSS

  • Inline CSS
  • Internal CSS
  • External CSS (Recommended)

Modern CSS Concepts

  • Flexbox
  • CSS Grid
  • Responsive Design
  • Media Queries
